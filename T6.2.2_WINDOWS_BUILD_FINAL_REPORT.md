### Windows构建工作流特点
```yaml
# 关键配置
- name: Build Tauri App
  run: |
    $env:TAURI_PRIVATE_KEY = "${{ secrets.TAURI_PRIVATE_KEY }}"
    $env:TAURI_KEY_PASSWORD = "${{ secrets.TAURI_KEY_PASSWORD }}"
    cargo tauri build --target ${{ matrix.target }}
```
